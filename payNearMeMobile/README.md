PayNearMeMobile Base
=====================

Make sure the `ionic` utility is installed:

```bash
$ sudo npm install -g ionic
```

Then run:

```bash
$ npm install
$ gulp init
$ ionic build ios
$ ionic emulate ios
```

Using SASS, just update the `./scss/ionic.app.scss` file
In a new terminal, watch for SASS compilatons
```bash
$ gulp watch
```
